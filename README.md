# RoomService
This is a plugin to expand functionality and events of Zeepkist and ZeepSDK into Zua.
On its own it does nothing, and will require a Zua script to use it.

## Dependencies
ZeepSDK [Mod.io](https://mod.io/g/zeepkist/m/zeepsdk) - [Github](https://github.com/donderjoekel/ZeepSDK/)<br>
